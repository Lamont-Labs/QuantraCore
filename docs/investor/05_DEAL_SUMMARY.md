## What We're Open To

### Path 1: IP Acquisition

**Structure:** Outright purchase of IP and codebase

**Includes:**
- All source code and specifications
- All documentation and test suites
- Model architectures and manifests
- Optional: QuantraVision bundle
- Optional: Founder consulting period

**Best For:** Funds or platforms wanting to own and develop internally

---

### Path 2: Licensing

**Structure:** Licensing agreement for institutional use

**Options:**
- **Engine License:** Access to deterministic engine and protocols
- **Full Stack License:** Engine + ApexLab + ApexCore + ApexDesk
- **White-Label:** Customized deployment with partner branding

**Best For:** Funds wanting to use without acquiring; platforms wanting embedded intelligence

---

### Path 3: Build-Out Partnership

**Structure:** Minority investment + engineering support

**Includes:**
- Capital investment for team and infrastructure
- Senior engineering resources
- Founder remains as architect/product lead
- Shared upside in commercialization

**Best For:** Partners who see long-term value and want to build together
